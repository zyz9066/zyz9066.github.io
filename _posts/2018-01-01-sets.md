# Objects as sets
### Set Theory
**What is a Set** - A set is a collection of obejects

|A set of books|
|A set of the colors in the rainbow|

### Set Properties
* The order of elements in a set is not important.
* A set with no elements is called an empty set.
* Two sets that have exactly the same elements are equal.
* Set cardinality is the number of elements in the set.

### Special Set Names
* M - the set of all natrual numbers (1, 2, 3, 4, ...).
* W - the set of whole numbers (0, 1, 2, 3, 4, ...).
* Z - is set of all integers (Z<sup>+</sup> are positive, Z<sup>-</sup> are negative).
* Q - the set of all rational numbers.
* R - the set of all real numbers.

### Subset Definition
If every element in set $$A$$ is also an element of set $$B$$, then $$A$$ is a subset of $$B$$, writen as: $$A\subseteq B$$\\
If $$A\subseteq B$$ and there is an element of $$B$$ that is not in $$A$$, then $$A$$ is a proper subset of $$B$$.\\
Example: $$A = \{2, 4, 6\}$$, $$B = \{2, 4, 6, 8, 10\}$$
### Set Elements and Subsets

# Set operations
* **Set Intersection**: $$A\cap B$$, $$\{x:x\in A\;and\;x\in B\}$$
* **Set Union**: $$A\cup B$$, $$\{x:x\in A\;or\;x\in B\;or\;both\}$$
* **Set Difference**: $$A-B$$, $$\{x:x\in A\;and\;x\notin B\}$$
* **Set Complement**: $$\overline {A}\;or\;A^C$$, $$\{x:x\notin A\}$$

### Universe of Discourse
Let $$U$$ represent the universal set, then the complement of a set $$A$$ is the set of all elements in $$U$$ that are not an element of $$A$$.\\
Example: $$U=\{Z\}$$, $$E=\{x\in Z\mid x=2i\}$$. Find the complement of $$E$$.
### Set Identity Laws
* Distributive Law: $$A\cap (B\cup C)=(A\cap B)\cup(A\cap C)$$
* Complement Law: $$A\cup \overline {A}=U$$
* Identity Law: $$A\cap U=A$$

**Prove: $$(A\cap B)\cup(A\cap \overline{B})=A$$**\\
$$(A\cap B)\cup(A\cap \overline{B})$$\\
$$=A\cap(B\cup \overline{B})$$ <- distributive law\\
$$=A\cap U$$ <- complement law\\
$$=A$$ <- identity law
# Power sets
### Definition
The Power Set is the set of all subsets for a particular set including the empty set.\\
The cardinality of a power set is $$2^n$$ where $$n$$ is the cardinality of the original set.\\
If $$|A|=3$$, then $$|\mathcal {P}(A)|=8$$\\
**Given the set $$A=\{1, 2, 3\}$$**\\
$$|\mathcal {P}(A)|=\{\emptyset, \{1\}, \{2\}, \{3\}, \{1, 2\}, \{1, 3\}, \{2, 3\}, \{1, 2, 3\}\}$$

# Sequences and Sums
### Sequences
* **Finite Sequence**: $$\{a_m,a_{m+1},a_{m+2},\;...,a_n\}$$
* **Infiinite Sequence**: $$\{a_m,a_{m+1},a_{m+2},\;...\}$$

Given the sequence: $$\{2^1,2^2,2^3,\;...\}$$, what is an explicit formula for this series?\\
We can represent each element as $$d_k = 2^k$$ for $$k\ge 1$$
* Increasing: $$a_k<a_{k+1}$$
* Non-decreasing: $$a_k\le a_{k+1}$$
* Decreasing: $$a_k>a_{k+1}$$
* Non-increasing: $$a_k\ge a_{k+1}$$

* Increasing: 1, 2, 3, 4
* Non-decreasing: 5, 6, 7, 7, 10
* Decreasing:9, 8, 4, 2, -2, -4
* Non-increasing: 4, 3, 3, 2, 1

### Special Sequences
* Geometric Sequence - each term is derived by taking the previous term and multiplying it by a common ratio. The sequence can be finite or infinite.
* Fibonacci sequence: 1, 1, 2, 3, 5, 8, 13, 21, ...

### Summations
Summation notation is used to express the sum of terms in a numerical sequence.\\
$$\sum\limits_{i=s}^n a_i=a_s+a_{s+1}+...+a_n$$\\
$$a_n=n^3$$ for $$n=1, 2, 3, 4$$\\
$$\sum\limits_{i=1}^4 i^3=1^3+2^3+3^3+4^3=100$$
### Separate the Last Term
$$\sum\limits_{k=m}^n a_k=a_m+a_{m+1}+a_{m+2}+...+a_n$$\\
$$\sum\limits_{k=1}^n (k+1)^2=\sum\limits_{k=1}^{n-1} (k+1)^2+(n+1)^2$$
### Change of Variables
$$\sum\limits_{j=1}^n (j+2)^3$$ substitute $$i=j+2$$, $$\sum\limits_{i=3}^{n+2} i^3$$
### Product Notation
$$\prod\limits_{i=1}^n i=1\times 2\times 3\times 4\times ... \times n$$

# Cartesian Products
### Definition
Cartesian products are a result of taking two sets, such as $$A$$ and $$B$$, and creating a list of ordered pairs.

$$A\; X\; B=\{(a,b)\mid a\in A, and\; b\in B\}$$\\
$$A$$: set of all students at a college.\\
$$B$$: set of all courses at the college.\\
$$A\times B$$: set of all student and course combinations.\\
$$|A|*|B|$$: the number of  ordered pairs

$$A=\{s_1,s_2\}$$, $$B=\{c_1,c_2,c_3\}$$\\
$$\{(s_1,c_1),(s_1,c_2),(s_1,c_3),(s_2,c_1),(s_2,c_2),(s_2,c_3)\}$$

* Ordered n-tuples are equal *iff* each correspongding pair of elements is equal.
* $$(a_1,a_2,...,a_n)=(b_1,b_2,...,b_n)$$ *iff* $$a_1=b_1$$, etc.
* A tuple with only two elements is called an ordered pair.
* An ordered pair of $$(a,b)=(c,d)$$ *iff* $$a=c$$ and $$b=d$$.
