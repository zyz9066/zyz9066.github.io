# What is markdown
* Plain text format
* Easy to read
* Easy to write
* Easy to learn

### How it works

```markdown
Headline
==========

Lorem ipsum dolor sit __amet__, consectetur adipisicing.
Dumque disngnsf, dupal. Ules asditn.

* Item 1
* Item 2
* Item 3
```

* Easy to parse
* Lots of support
* Many extensions
